logic-on-canvas
===============

## Build
コードの変更して、適用するときにビルド

    # build all
    >> neko build.n
    # build codeiq
    neko build.n codeiq
    # build raspberrypi
    neko build.n raspberrypi

## codeiqサーバ起動
    
    # codeiqサーバに移動
    >> cd codeiq
    >> node server.js

http://localhost:8888
でサーバを起動する。
